# Code of Conduct

This project is small and focused, but I'm committed to keeping it a respectful, inclusive space for everyone who wants to contribute or use the library.

## Expected Behavior

- **Be kind and constructive** in all interactions
- **Assume good intent** and give people the benefit of the doubt
- **Provide helpful feedback** when reviewing code or discussing ideas
- **Respect different perspectives** on technical decisions

## Unacceptable Behavior

I won't tolerate:

- Harassment, threats, or abusive language
- Discrimination based on personal characteristics
- Sharing others' private information (doxxing)
- Sustained disruption of discussions

## Scope

This code of conduct applies to all project spaces, including issues, pull requests, discussions, and other community channels.

## Enforcement

If someone violates this code of conduct:

1. I'll review the report promptly (usually within 48 hours)
2. Depending on severity, I may:
   - Issue a private warning
   - Remove offensive content
   - Temporarily or permanently ban the person from the project

I'll always explain my reasoning to those involved.

## Reporting

Report code of conduct violations by emailing: **report@shaitern.dev**

All reports will be reviewed within 48 hours and handled discreetly.
