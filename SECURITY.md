# Security Policy

## Supported Versions

Only the latest major version of @shaisrc/tty is actively supported with security updates. Please update to the latest release for security patches.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately to the maintainer:

- Email: report@shaitern.dev

Do **not** file public issues for security problems. We will respond within 48 hours and coordinate a fix and disclosure timeline with you.

## Disclosure Policy

- We aim to respond to all security reports within 48 hours.
- Fixes will be released as soon as possible after confirmation.
- Credit will be given to reporters unless anonymity is requested.
